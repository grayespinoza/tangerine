# Tangerine for [Godot](https://github.com/godotengine/godot)

## Usage

### Interface Theme

1. Open _Godot_
2. Go to _Editor_ → _Editor Settings_ → _Interface_ → _Theme_
3. Use
   - Base Color:
     - Dark: `#1b1b1b`
     - Light: `#d1d1d1`
   - Accent Color: `#fe640b`
   - Contrast: `0.25`
   - Icon Saturation: `2.0`
